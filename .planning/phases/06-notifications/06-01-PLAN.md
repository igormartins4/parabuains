### Task 4 (auto): EmailTransport

Create `apps/api/src/modules/notifications/email.transport.ts`:

```typescript
export class EmailTransport {
  private resend: Resend;
  
  constructor(apiKey: string, private fromEmail: string) {
    this.resend = new Resend(apiKey);
  }

  async sendBirthdayReminder(params: {
    to: string;
    recipientName: string;
    birthdayPersonName: string;
    daysUntil: number; // 0 = today, 1, 3, 7
    birthdayDate: string; // "MM-DD"
  }): Promise<void>;

  async sendWallMessageNotification(params: {
    to: string;
    recipientName: string;
    senderName: string | null; // null = anonymous
    profileUsername: string;
  }): Promise<void>;

  async sendFriendshipAcceptedNotification(params: {
    to: string;
    recipientName: string;
    friendName: string;
    friendUsername: string;
  }): Promise<void>;
}
```

Simple plain-text email via `resend.emails.send()`. Subject lines in Portuguese:
- Birthday reminder: `"Lembrete: aniversário de {name} em {days} dias"` (or `"Hoje é o aniversário de {name}!"` for day 0)
- Wall message: `"Você recebeu uma mensagem no seu mural"`
- Friendship accepted: `"{friendName} aceitou seu pedido de amizade"`

**Done criteria:** File created; compiles without error.

### Task 5 (auto): NotificationService

Create `apps/api/src/modules/notifications/notification.service.ts`:

```typescript
export class NotificationService {
  constructor(
    private prefsRepo: INotificationPreferencesRepository,
    private logRepo: INotificationLogRepository,
    private emailTransport: EmailTransport,
    private userRepo: { findById(id: string): Promise<{ email: string; displayName: string } | null> },
  ) {}

  async getPreferences(userId: string): Promise<NotificationPreference[]>;
  async upsertPreference(userId: string, channel: 'email' | 'push', daysBefore: number[], enabled: boolean): Promise<NotificationPreference>;
  
  // Called by BullMQ worker for wall message events
  async handleWallMessagePosted(profileId: string, authorId: string | null): Promise<void>;
  
  // Called by BullMQ worker for friendship accepted events
  async handleFriendshipAccepted(recipientId: string, actorId: string): Promise<void>;
}
```

`handleWallMessagePosted`: get profile user, get profile email prefs, if email enabled → send email.  
`handleFriendshipAccepted`: get requester, get their email prefs, if email enabled → send friendship accepted email.

Both methods log to `notification_log` with `reminderType = 'Dday'` for event-based notifications.

**Done criteria:** File created; compiles without error.
